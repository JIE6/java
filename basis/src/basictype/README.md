# 基本类型与运算符
## *Int, *Char, *Double, *Boolean， *AutoConversion
### 八大基本类型(^表示次方，例如2^7表示2的七次方)

| 类型 | 占用字节 | 占用位数 | 取值范围 |
|:----:|:----:|:----:|:----:|
| byte | 1 | 8 | -2^7 ~ 2^7-1 |
| short | 2 | 16 | -2^15 ~ 2^15-1 |
| int | 4 | 32 | -2^31 ~ 2^31-1 |
| long | 8 | 64 | -2^63 ~ 2^63-1 |
| char | 2 | 16 | 0 ~ 65535 |
| float | 4 | 32 | 2^-149 ~ 2^128-1 |
| double | 8 | 64 | 2^-1074 ~ 2^1024-1 |
| boolean | 1 | 8 | true~false |
